Bundled Unicode chess-capable fonts used by the Unicode piece styles.

Included files:
- DejaVuSans.ttf (DejaVu Fonts)
- NotoSansSymbols2-Regular.ttf (Google Noto Fonts)
- Quivira.otf (Quivira Unicode Font)

License/source notes:
- DejaVu Fonts: permissive license based on Bitstream Vera + DejaVu additions
  https://dejavu-fonts.github.io/License.html
- Noto Sans Symbols 2: SIL Open Font License 1.1 (Google Noto)
  https://notofonts.github.io/
- Quivira: official site states public domain since 2019
  http://quivira-font.com/
